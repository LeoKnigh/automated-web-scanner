Специальность: Информационная безопасность телекоммуникационных систем

## Описание
Система для автоматического сканирования веб-приложений на наличие известных уязвимостей с использованием open-source инструментов.

## Технологии
- Python 3.9+
- Docker
- GitLab CI/CD
- OpenVAS, OWASP ZAP
